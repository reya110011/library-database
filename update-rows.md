### update statement
```sql
update
  table_name
  set
  column_name=value
  where
  column_name=value
  and
    column_name=value
  ```


  ### example
  ```sql
  update
      books
set
  checked=1
  where
  title='sample things'
  and
  publicationyear=2018
  ```
